layout

alignment

all elements should be vertically aligned to the top of their containers, which ensures consistency across the layout of horizontally adjacent content blocks.

responsiveness

the page should be scalable across differently-sized viewports: utilizing the display mode inline-block and preferring to define containers using max-width will allow for automatic resizing of each element as users scale their browser windows.

structure

each major section should be divided into smaller discrete blocks. these may be defined using div containers or semeantic elements as the case may allow, but in all cases should be visually compatible with the responsive direction outlined previously.
